# Life Map Website

Interactive Life Map showing 5000 weeks, past weeks, birth week, life expectancy markers, world events, Chinese festivals, and personal/travel events.

## How it works

- Personal events are loaded from `src/data/events.json`.
- World events and Chinese festivals can be filtered using toggles.
- Grid shows past weeks, future weeks, and life expectancy markers.

## Deployment

1. Build the project: `npm run build`
2. Upload `build` folder to GitHub Pages
